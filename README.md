# HW4 Checker

A Tesla VIN hardware checker — paste a VIN, get a Hardware 4 verdict instantly, fully offline.

**Status:** In development

**Live app (when deployed):** https://manaux.github.io/HW4-checker/

**Repo:** https://github.com/manaux/HW4-checker

---

Built with React + Vite + TypeScript. All decoding and verdict logic runs in the browser — your VIN never leaves your device.
